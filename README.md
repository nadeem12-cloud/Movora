## 📌 What is Movora?

**Movora** is a budget-based car discovery tool built with Streamlit. Enter your price range and instantly browse matching vehicles from a rich Kaggle dataset — no scrolling through endless listings required.

The dataset is packed with detailed car attributes including mileage, engine specs, and fuel type, making it a solid foundation for deeper features in future versions.

---

## ✨ Features

- 💰 **Price Range Filter** — Set your min and max budget to instantly filter matching cars
- 📊 **Rich Dataset** — Powered by a comprehensive Kaggle car dataset with detailed specs
- ⚡ **Instant Results** — Clean tabular view of matching vehicles with key attributes
- 🖥️ **Simple UI** — No clutter, just the info you need

## ⚙️ Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/nadeem12-cloud/Movora.git
cd Movora
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

Or visit the live app: **[movora.streamlit.app](https://movora.streamlit.app)**
